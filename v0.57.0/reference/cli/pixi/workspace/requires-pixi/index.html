    About#

    Commands to manage the pixi minimum version requirement

    Usage#

    pixi workspace requires-pixi [OPTIONS] <COMMAND>

    Subcommands#

    Command Description
    get Get the pixi minimum version requirement
    set Set the pixi minimum version requirement
    unset Remove the pixi minimum version requirement
    verify Verify the pixi minimum version requirement

    Global Options#

    • --manifest-path <MANIFEST_PATH>
      The path to pixi.toml, pyproject.toml, or the workspace directory
