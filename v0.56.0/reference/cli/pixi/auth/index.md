# `[pixi](../) auth`

## About

Login to prefix.dev or anaconda.org servers to access private channels

## Usage

```text
pixi auth <COMMAND>

```

## Subcommands

| Command             | Description                                        |
| ------------------- | -------------------------------------------------- |
| [`login`](login/)   | Store authentication information for a given host  |
| [`logout`](logout/) | Remove authentication information for a given host |
