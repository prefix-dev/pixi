# `[pixi](../) completion`

## About

Generates a completion script for a shell

## Usage

```text
pixi completion --shell <SHELL>

```

## Options

- [`--shell (-s) <SHELL>`](#arg---shell) The shell to generate a completion script for

  **required**: `true`

  **options**: `bash`, `elvish`, `fish`, `nushell`, `powershell`, `zsh`
