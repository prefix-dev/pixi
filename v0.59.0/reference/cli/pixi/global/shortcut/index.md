# `[pixi](../../) [global](../) shortcut`

## About

Interact with the shortcuts on your machine

## Usage

```text
pixi global shortcut <COMMAND>

```

## Subcommands

| Command             | Description                                         |
| ------------------- | --------------------------------------------------- |
| [`add`](add/)       | Add a shortcut from an environment to your machine. |
| [`remove`](remove/) | Remove shortcuts from your machine                  |
